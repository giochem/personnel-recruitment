# Personnel recruitment web

## Demo

[https://personnel-recruitment.herokuapp.com/](https://personnel-recruitment.herokuapp.com/)

## Usage

Rename the .envexample to .env and your variable

### Install dependencies

```
# Backend deps
npm install

# Frontend deps
cd frontend
npm install
```

### Run Server

```
npm run server
```

### RUN Client

```
npm run client
```

### Run Fullstack

```
npm run dev
```
